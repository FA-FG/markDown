Creating a file using the terminal is a fundamental skill for anyone working with command-line interfaces, whether you're developing software, managing systems, or automating tasks. Hereâ€™s how you can create a file and manage its contents directly from the terminal.

## Steps:

1. **Creat** new folder
2. **Navigate** to the folder
3. **Create** the new file .

### 1. Create a new folder of navigate to existing one

`mkdir example`

### 2. Navigate to the folder just created

`cd example`

### 3. Create a new file

`touch newFile.txt`

> Tip: you can edit you files using text editors ex. vim or a code editor ex. vsconde

**_Example_**:

```
mkdir myProgram
cd myProgram
touch index.html

```

> Tip: you can create multiple files at ounce

> > `touch index.html style.css app.js`
